# Project Overview

This project was completed as part of the **Mathematics for Machine Learning** module.

The assignment explores three fundamental machine learning concepts:

- Probability Distributions using the Expectation-Maximization (EM) algorithm
- Bayesian Probability for sentiment analysis
- Gradient Descent for parameter optimization in linear regression

The project combines mathematical derivations with Python implementations to demonstrate both theoretical understanding and practical application.

---

# Project Components

## Part 1 – Probability Distributions

Implemented the **Expectation-Maximization (EM) algorithm** from scratch to separate two Gaussian distributions representing different populations based on height data.

Key tasks included:

- Gaussian mixture modelling
- Expectation Step
- Maximization Step
- Parameter updates
- Posterior probability prediction
- Log-likelihood tracking

---

## Part 2 – Bayesian Probability

Implemented **Bayes' Theorem** using the IMDb Movie Reviews dataset.

The implementation computes:

- Prior Probability
- Likelihood
- Marginal Probability
- Posterior Probability

using selected positive and negative sentiment keywords.

---

## Part 3 – Manual Gradient Descent

Performed manual calculations of Gradient Descent for a linear regression model.

Tasks included:

- Prediction calculations
- Mean Squared Error (MSE)
- Gradient derivation
- Parameter updates
- Iterative optimization

---

## Part 4 – Gradient Descent in Python

Converted the manual Gradient Descent calculations into Python.

The implementation includes:

- Gradient computation
- Parameter updates
- Prediction generation
- Error tracking
- Visualization of convergence using Matplotlib
